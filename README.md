### Run unit test
* Pull the latest code
* Install gem using bundle
```bash
$ bundle
```
* Run unit test
```bash
$ rspec spec/models/user_spec.rb && rspec spec/models/business_spec.rb
```
* Run cucumber test scenario on selenium
```bash
$ cucumber features/user_page.feature 
```
